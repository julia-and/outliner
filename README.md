# ol - Outline Note Taking

## Overview

1. An outline-first note taking application.
2. Local-first architecture.
3. Uses Yjs, Dexie, Tiptap, React, Typescript.
4. Focus on high performance keyboard navigable interface.
5. Two modes in the outline: navigation and edit.
